# Number Guesser Bot

## Description
Number Guesser Bot is a Python game where the computer tries to guess the number you are thinking of. It uses a smart guessing algorithm (binary search) to efficiently narrow down your number based on your hints: "higher", "lower", or "correct". The game includes multiple difficulty levels and a replay feature to make it interactive and fun.

This project is ideal for beginners learning Python while building a practical and engaging program.

---

## Features
- Two difficulty levels:
  - **Easy:** Numbers between 1 and 100  
  - **Hard:** Numbers between 1 and 500
- Efficient guessing using **binary search logic**
- Interactive responses for user input
- Tracks the number of attempts
- Replay option to play multiple rounds
- Handles invalid inputs gracefully
